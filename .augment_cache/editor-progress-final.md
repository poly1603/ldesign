## 🌐 使用方法

### 快速开始
```bash
# 进入项目目录
cd packages/editor

# 启动开发服务器
pnpm dev

# 访问示例页面
# http://localhost:3000/examples/basic.html

# 运行测试
pnpm test
```

### 基础使用
```typescript
import { LDesignEditor } from '@ldesign/editor'

const editor = new LDesignEditor({
  container: '#editor',
  content: '<p>Hello World!</p>',
  theme: 'default'
})

editor.init()
```
