# 组件总览

LDesign 提供了丰富的组件，涵盖了常见的 UI 需求。

## 基础组件

### Button 按钮
常用的操作按钮。

```vue
<template>
  <l-button type="primary">
    主要按钮
  </l-button>
</template>
```

### Icon 图标
语义化的矢量图形。

```vue
<template>
  <l-icon name="home" />
</template>
```

## 布局组件

### Grid 栅格
24 栅格系统，迅速简便地创建布局。

### Layout 布局
协助进行页面级整体布局。

### Space 间距
设置组件之间的间距。

## 导航组件

### Menu 导航菜单
为页面和功能提供导航的菜单列表。

### Breadcrumb 面包屑
显示当前页面在系统层级结构中的位置，并能向上返回。

### Pagination 分页
采用分页的形式分隔长列表，每次只加载一个页面。

## 数据录入

### Input 输入框
通过鼠标或键盘输入内容，是最基础的表单域的包装。

### Select 选择器
下拉选择器。

### Checkbox 多选框
多选框。

### Radio 单选框
单选框。

### Switch 开关
开关选择器。

### DatePicker 日期选择器
输入或选择日期的控件。

### TimePicker 时间选择器
输入或选择时间的控件。

### Upload 上传
文件选择上传和拖拽上传控件。

### Form 表单
具有数据收集、校验和提交功能的表单，包含复选框、单选框、输入框、下拉选择框等元素。

## 数据展示

### Table 表格
展示行列数据。

### Tag 标签
进行标记和分类的小标签。

### Progress 进度条
展示操作的当前进度。

### Tree 树形控件
多层次的结构列表。

### Tooltip 文字提示
简单的文字提示气泡框。

### Popover 气泡卡片
点击/鼠标移入元素，弹出气泡式的卡片浮层。

### Badge 徽标数
图标右上角的圆形徽标数字。

### Avatar 头像
用来代表用户或事物，支持图片、图标或字符展示。

### Card 卡片
通用卡片容器。

### Carousel 走马灯
在有限空间内，循环播放同一类型的图片、文字等内容。

### Collapse 折叠面板
可以折叠/展开的内容区域。

## 反馈组件

### Alert 警告提示
警告提示，展现需要关注的信息。

### Loading 加载中
加载数据时显示动效。

### Message 全局提示
全局展示操作反馈信息。

### MessageBox 弹框
模拟系统的消息提示框而实现的一套模态对话框组件，用于消息提示、确认消息和提交内容。

### Notification 通知提醒框
悬浮出现在页面角落，显示全局的通知提醒消息。

### Popconfirm 气泡确认框
点击元素，弹出气泡式的确认框。

### Drawer 抽屉
屏幕边缘滑出的浮层面板。

### Dialog 对话框
在保留当前页面状态的情况下，告知用户并承载相关操作。

## 其他

### BackTop 回到顶部
返回页面顶部的操作按钮。

### Divider 分割线
区隔内容的分割线。

### Image 图片
图片容器，在保留原生img的特性下，支持懒加载，自定义占位、加载失败等。

### InfiniteScroll 无限滚动
滚动至底部时，加载更多数据。

### Skeleton 骨架屏
在需要等待加载内容的位置设置一个骨架屏，某些场景下比 Loading 的视觉效果更好。

## 设计规范

所有组件都遵循统一的设计规范：

- **一致性**：相同类型的组件保持一致的外观和行为
- **可访问性**：支持键盘导航和屏幕阅读器
- **响应式**：适配不同屏幕尺寸
- **主题化**：支持自定义主题和暗黑模式
