## 🚀 快速体验

### 在线演示

所有示例都提供在线演示，您可以直接在浏览器中体验：

- [基础应用演示](https://ldesign-basic-demo.netlify.app)
- [企业级应用演示](https://ldesign-enterprise-demo.netlify.app)
- [移动端应用演示](https://ldesign-mobile-demo.netlify.app)

### 本地运行

克隆项目并运行示例：

```bash
# 克隆项目
git clone https://github.com/ldesign-org/ldesign.git
cd ldesign

# 安装依赖
pnpm install

# 运行基础示例
cd examples/basic-app
pnpm dev

# 运行企业级示例
cd examples/enterprise-app
pnpm dev
```
