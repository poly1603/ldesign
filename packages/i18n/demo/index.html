基础翻译功能

简单翻译

i18n.t('hello')

带参数翻译

i18n.t('welcome', { name: '张三' })

嵌套键翻译

i18n.t('user.profile.name')

高级功能

复数化

i18n.t('item', { count: 1 }) / i18n.t('item', { count: 5 })

批量翻译

i18n.translateBatch(['hello', 'welcome'])

键存在性检查

i18n.exists('hello') / i18n.exists('nonexistent')

格式化功能

数字格式化

formatNumber(1234.56, 'zh-CN')

货币格式化

formatCurrency(1234.56, { locale: 'zh-CN', currency: 'CNY' })

日期格式化

formatDate(new Date(), 'zh-CN')
