@ldesign/chart

基于 ECharts 的通用图表组件库

功能概览

@ldesign/chart 是一个基于 ECharts 的通用图表组件库，提供简化的 API 和数据驱动的设计理念。

🚀 框架无关

可在 React、Vue、原生 JavaScript 等任意环境中使用

📊 多种图表

支持折线图、柱状图、饼图、散点图、面积图等

🎨 主题系统

内置多种主题，支持自定义主题

📱 响应式

自动适应容器大小变化

⚡ 高性能

支持大数据量渲染，优化内存使用

🔧 TypeScript

完整的类型定义，开发体验更佳
