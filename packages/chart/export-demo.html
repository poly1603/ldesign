图表导出功能演示

展示 @ldesign/chart 强大的导出功能，支持多种格式导出

导出演示

下面的图表支持导出为多种格式，点击按钮体验导出功能：

导出功能特性

🖼️ 图片导出

支持导出为 PNG 和 SVG 格式，可自定义尺寸、质量和背景色

📄 PDF 导出

将图表导出为 PDF 文档，适合报告和文档嵌入

📊 数据导出

支持导出为 CSV、Excel 和 JSON 格式，便于数据分析

🎨 自定义选项

丰富的导出选项，包括文件名、尺寸、质量等设置

使用示例

以下是如何在代码中使用导出功能：

// 导出为 PNG 图片 await chart.downloadExport('png', 'my-chart') // 导出为 PDF await chart.downloadExport('pdf', 'my-chart') // 导出数据为 CSV await chart.downloadExport('csv', 'my-data') // 自定义导出选项 await chart.export({ format: 'png', filename: 'custom-chart', width: 800, height: 600, quality: 0.9, backgroundColor: 'white' }) // 获取图片数据 URL（不下载） const imageDataURL = await chart.exportImage('png')
