技术实现细节

1. NodeLayoutHelper 工具类：

  • calculateLayout() - 计算最佳布局位置
  • 支持垂直、水平、仅图标、仅文本四种布局模式
  • 自动检测是否需要调整节点大小

2. CSS 样式优化：

  • node-layout.css - 统一的节点布局样式
  • 响应式设计支持移动端
  • 高对比度模式和无障碍支持

3. 节点更新：

  • StartNode, EndNode - 圆形节点布局优化
  • ProcessNode, UserTaskNode, ServiceTaskNode - 矩形节点布局优化
  • ConditionNode - 菱形节点布局优化
  • CustomMaterialNode - 自定义物料节点布局优化
