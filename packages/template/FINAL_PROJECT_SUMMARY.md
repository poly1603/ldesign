## 🚀 使用方式

### 组件方式
```vue
<template>
  <TemplateRenderer
    :template-name="templateName"
    :device="deviceType"
    :config="templateConfig"
  />
</template>
```

### Hook方式
```typescript
const {
  currentTemplate,
  availableTemplates,
  switchTemplate,
  loading
} = useTemplate({
  autoScan: true,
  enableHMR: true
})
```
