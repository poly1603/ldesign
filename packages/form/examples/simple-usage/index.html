@ldesign/form - 简单使用示例

🎯 基础表单

最简单的表单配置，只需要定义字段即可：

<template> <DynamicForm v-model="formData" :options="formOptions" /> </template> <script setup> import { ref } from 'vue' import { DynamicForm } from '@ldesign/form' const formData = ref({}) const formOptions = { fields: [ { name: 'name', title: '姓名', component: 'FormInput', required: true }, { name: 'email', title: '邮箱', component: 'FormInput', props: { type: 'email' } }, { name: 'age', title: '年龄', component: 'FormInput', props: { type: 'number' } } ] } </script>

🎨 主题样式

支持多种主题样式，只需要设置layout.theme：

const formOptions = { fields: [...], layout: { theme: 'bordered', // 'default' | 'bordered' columns: 2, // 列数 label: { position: 'left', // 标签位置 showColon: true, // 显示冒号 gap: 12 // 标签间距 } } }

🔄 条件显示

使用showWhen配置实现字段的条件显示：

const formOptions = { fields: [ { name: 'country', title: '国家', component: 'FormSelect', props: { options: [ { label: '中国', value: 'china' }, { label: '美国', value: 'usa' } ]} }, { name: 'province', title: '省份', component: 'FormInput', showWhen: { field: 'country', value: 'china' } // 只有选择中国时才显示 } ] }

📱 响应式布局

支持默认行数和展开/收起功能：

const formOptions = { fields: [...], // 很多字段 layout: { defaultRows: 2, // 默认显示2行 button: { position: 'follow-last-row', // 按钮跟随最后一行 align: 'right' // 右对齐 } } }
