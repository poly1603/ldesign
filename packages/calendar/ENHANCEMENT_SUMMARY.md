## 🎮 使用方法

### 基本使用
```javascript
import { Calendar } from '@ldesign/calendar'
import '@ldesign/calendar/dist/index.css'

const calendar = new Calendar('#calendar', {
  locale: 'zh-CN',
  showLunar: true,
  theme: 'default'
})
```

### 交互功能
- **右键点击日期** → 显示上下文菜单
- **右键点击事件** → 编辑/删除/复制
- **双击日期** → 查看详细信息
- **拖拽事件** → 移动到其他日期
- **键盘导航** → 方向键、空格键、Home键

### 视图切换
```javascript
calendar.changeView('desk')  // 切换到台历视图
calendar.changeView('month') // 切换到月视图
```
