# @ldesign/cropper 演示指南

## 🎉 演示项目已成功创建！

我已经成功创建了一个完整的 Vite + Vue 3 演示项目，展示了 @ldesign/cropper 图片裁剪插件的四种不同使用方式。

## 🚀 快速启动

### 1. 安装依赖
```bash
cd packages/cropper/demo
pnpm install
```

### 2. 启动开发服务器
```bash
pnpm run dev
```

### 3. 访问演示
打开浏览器访问：http://localhost:3000

### 项目结构
```

├── src/
│   ├── views/           # 四个演示页面
```
