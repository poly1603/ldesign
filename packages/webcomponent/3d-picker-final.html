📋 使用说明

  1. 必须添加上面的 CSS 样式 - 当前构建系统存在问题，3D样式未被正确编译
  2. 设置 enable-3d 属性 - 启用3D效果
  3. 配置 CSS 变量 - 调整透视距离、半径、旋转角度等参数
  4. 可选 show-mask - 添加上下渐变遮罩增强效果

关键 CSS 变量

变量名 说明 推荐值
--ldesign-picker-3d-perspective 透视距离 600-1000px
--ldesign-picker-3d-radius 圆柱半径 100-150px
--ldesign-picker-3d-step-deg 步进角度 15-25deg
--ldesign-picker-3d-scale-min 最小缩放 0.6-0.85
--ldesign-picker-3d-scale-max 最大缩放 1.05-1.2
