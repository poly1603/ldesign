## 核心模块

### PdfApi
主要的PDF处理API，提供文档加载、页面渲染等功能。

### PdfEngine
底层PDF引擎，处理PDF.js集成和优化。

### 缓存系统
智能缓存系统，提升PDF处理性能。

### Worker系统
多线程处理支持，避免阻塞主线程。
