🚀 快速开始

安装PDF阅读器插件：

npm install @ldesign/pdf-reader pdfjs-dist

基本使用方法：

import { createPDFReader } from '@ldesign/pdf-reader'

const pdfReader = createPDFReader({
  container: document.getElementById('pdf-container'),
  showToolbar: true,
  showThumbnails: true,
  theme: 'light'
})

await pdfReader.loadDocument('/path/to/document.pdf')

第二页内容

📖 API文档

createPDFReader(options)

创建PDF阅读器实例的工厂函数。

参数

  • container - 容器元素或选择器
  • showToolbar - 是否显示工具栏（默认：true）
  • showThumbnails - 是否显示缩略图（默认：true）
  • theme - 主题设置（'light' | 'dark' | 'auto'）
  • initialScale - 初始缩放比例（默认：1.0）
  • enableSearch - 是否启用搜索（默认：true）

返回值

返回一个PDFReader实例，包含以下方法：

  • loadDocument(src) - 加载PDF文档
  • goToPage(pageNumber) - 跳转到指定页面
  • previousPage() - 上一页
  • nextPage() - 下一页
  • setScale(scale) - 设置缩放比例
  • zoomIn() - 放大
  • zoomOut() - 缩小
  • fitWidth() - 适合宽度
  • fitPage() - 适合页面
  • search(query) - 搜索文本
  • clearSearch() - 清除搜索
  • getState() - 获取当前状态
  • destroy() - 销毁实例

第三页内容

🎯 事件系统

PDF阅读器支持丰富的事件系统，可以监听各种状态变化：

// 监听文档加载完成
pdfReader.on('document-loaded', (info) => {
  console.log('文档已加载:', info)
})

// 监听页面变化
pdfReader.on('page-changed', (pageNumber) => {
  console.log('当前页面:', pageNumber)
})

// 监听缩放变化
pdfReader.on('scale-changed', (scale) => {
  console.log('缩放比例:', scale)
})

// 监听搜索结果
pdfReader.on('search-result', (results) => {
  console.log('搜索结果:', results)
})

// 监听错误
pdfReader.on('error', (error) => {
  console.error('发生错误:', error)
})
