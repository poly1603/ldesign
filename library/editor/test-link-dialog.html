🔗 链接插入功能测试

智能识别选中文本，优雅的对话框界面

📝 场景一

直接点击链接按钮，需要同时输入链接文本和URL

✨ 场景二

先选中文本再点击链接按钮，只需输入URL即可

🚀 智能功能

自动补全 https:// 协议，支持 Enter 快捷键确认

功能特点

  • ✓ 智能识别 - 自动检测是否有选中文本，智能显示输入框
  • ✓ 优雅界面 - 美观的对话框设计，流畅的动画效果
  • ✓ 输入验证 - 实时验证输入，确保链接格式正确
  • ✓ 协议补全 - 自动为缺少协议的URL添加 https://
  • ✓ 快捷操作 - 支持 Enter 确认，ESC 取消

测试步骤

  • 1 直接点击工具栏的链接按钮，在对话框中输入文本和URL
  • 2 选中编辑器中的文本（如 "LDesign Editor"），再点击链接按钮
  • 3 测试只输入域名（如 example.com），系统会自动补全 https://
  • 4 尝试使用键盘快捷键：Enter 确认，ESC 取消
