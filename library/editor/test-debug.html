🔍 富文本编辑器调试测试

📋 测试步骤：

  1. 打开浏览器的开发者工具（F12）
  2. 切换到 Console 标签页
  3. 在编辑器中输入一些文字，例如："测试文字"
  4. 选中文字
  5. 点击下面的测试按钮
  6. 查看控制台的日志输出

⚠️ 重要：请截图或复制控制台的所有日志信息发给我！

📌 使用说明：

这是一个调试页面，需要与开发服务器一起使用。

步骤：

  1. 在终端运行: npm run dev
  2. 访问开发服务器提供的URL（通常是 http://localhost:5174）
  3. 导航到这个测试页面或在示例页面中测试
  4. 查看控制台输出，找出问题所在
