✨ 功能特性

🔎 模糊搜索

支持模糊匹配模式，让搜索更加灵活智能

  • 自动匹配相似文本
  • 忽略中间字符
  • 智能排序结果

⚡ 实时预览

输入即搜索，实时显示匹配结果

  • 300ms 防抖优化
  • 高亮显示所有匹配
  • 显示匹配计数

🔄 批量替换

一键替换所有匹配项，支持确认预览

  • 替换前确认
  • 显示影响范围
  • 支持撤销操作

🎯 精确控制

多种搜索选项，满足不同需求

  • 区分大小写
  • 全字匹配
  • 正则表达式

📝 搜索历史

自动保存搜索历史，方便快速重用

  • 保存最近10条
  • 本地存储持久化
  • 快速选择历史项

⌨️ 快捷键

丰富的键盘快捷键，提升操作效率

  • Ctrl+F 打开查找
  • Enter 查找下一个
  • Shift+Enter 查找上一个
  • Ctrl+H 切换到替换
  • Ctrl+Shift+Enter 全部替换
  • ESC 关闭对话框

📖 示例文本

编辑器中已预加载了一些示例文本，您可以尝试以下操作：

  • 按 Ctrl+F 打开查找对话框
  • 尝试搜索 "编辑器" 或 "文本"
  • 启用模糊搜索，输入 "bjq" 可以找到 "编辑器"
  • 使用正则表达式搜索 "\d+" 查找所有数字
  • 选中文本后按 Ctrl+F，自动填充搜索框
