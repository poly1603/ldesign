✅ 动画检查清单

  • 遮罩层淡入效果流畅（0.2s fadeIn）
  • 对话框上滑效果流畅（0.3s slideUp）
  • 关闭时动画反向播放
  • 背景模糊效果（backdrop-filter）
  • 动画时间一致（fadeIn 0.2s, slideUp 0.3s）
  • 点击遮罩层可关闭对话框
  • 按 ESC 键可关闭对话框
  • 动画结束后元素正确移除
