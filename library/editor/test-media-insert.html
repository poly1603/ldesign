🎬 媒体插入功能测试

支持图片、视频、音频的本地上传和网络地址插入

✨ 功能特性

  • 📸 图片插入：支持本地上传和网络图片地址
  • 🎥 视频插入：支持常见视频格式
  • 🎵 音频插入：支持音频文件播放
  • 📁 多选支持：图片支持批量上传
  • 🌐 网络地址：通过URL插入媒体
  • 🎨 统一界面：使用相同的下拉框动画

📝 富文本编辑器

独立测试按钮

🎯 使用说明

  • 📸
    插入图片
    点击工具栏的图片按钮，选择本地上传或输入网络图片地址。支持JPG、PNG、GIF等格式。
  • 🎥
    插入视频
    点击工具栏的视频按钮，上传本地视频或输入视频URL。支持MP4、WebM等格式。
  • 🎵
    插入音频
    点击工具栏的音频按钮，选择音频文件或输入音频地址。支持MP3、WAV等格式。
  • 📁
    批量上传
    图片支持多选上传，按住Ctrl或Shift键可以选择多个文件。
  • 🌐
    网络地址
    支持直接输入媒体文件的网络地址，包括CDN链接、图床链接等。
