🎨 LDesign Editor - 统一下拉框体验

所有下拉功能使用统一的动画效果，提供一致的用户体验

✨ 统一动画系统

  • 📍 打开动画：0.2秒渐入 + 缩放效果，使用 cubic-bezier(0.4, 0, 0.2, 1) 缓动
  • 📍 关闭动画：0.15秒渐出 + 缩放效果，快速响应用户操作
  • 📍 交互反馈：悬停时选项右移2px，点击时有视觉反馈
  • 📍 键盘支持：ESC键快速关闭，无需点击
  • 📍 智能定位：自动调整位置避免超出屏幕
📝 富文本编辑器
