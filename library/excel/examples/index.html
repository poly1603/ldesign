LDesign Excel Editor

基于Web的现代化Excel表格编辑器

📊 基础示例

这是一个基本的Excel编辑器示例，支持单元格编辑、数据输入等基础功能。

准备就绪

✨ 功能特性

🎯 单元格编辑

支持双击或按Enter键编辑单元格，支持多种数据类型

⌨️ 键盘导航

使用方向键在单元格间导航，支持Tab键快速移动

📁 文件操作

支持导入导出Excel文件，兼容.xlsx和.xls格式

🎨 主题定制

支持浅色和深色主题，可自定义样式和颜色

📱 响应式设计

适配各种屏幕尺寸，在移动设备上也能良好使用

🔧 TypeScript支持

完整的TypeScript类型定义，提供更好的开发体验

📖 使用说明

基本操作：

  • 单击选中单元格
  • 双击或按Enter键开始编辑
  • 使用方向键在单元格间移动
  • 按Escape键取消编辑
  • 按Delete键清空单元格内容

文件操作：

  • 点击"导入Excel"按钮选择本地Excel文件
  • 点击"导出Excel"按钮下载当前表格
  • 支持.xlsx和.xls格式

数据类型：

  • 自动识别数字、文本、布尔值和日期
  • 支持公式计算（开发中）
  • 支持单元格格式化（开发中）
