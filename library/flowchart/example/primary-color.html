🎨 流程图主题色配置

✨ 主题色功能说明

• 主题色会在鼠标悬停时应用到边线上，让交互更加统一

• 边线和箭头会变为主题色，并显示对应的发光效果

• 标签不会缩放移动，只会加粗文字，保持稳定的用户体验

• 支持任意颜色值（HEX、RGB、HSL等格式）

// 创建流程图时配置主题色
const flowChart = new FlowChart({
  container: '#flowchart',
  width: 800,
  height: 600,
  primaryColor: '#2196f3',  // 设置主题色
  enableZoom: true,
  enablePan: true
});

// 动态修改主题色
function changePrimaryColor(color) {
  const container = document.querySelector('#flowchart');
  container.style.setProperty('--flowchart-primary-color', color);
}

// 主题色会应用到：
// - 边线hover时的颜色
// - 箭头hover时的颜色
// - 发光效果的颜色
