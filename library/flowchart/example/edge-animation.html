✨ 边的动画效果特性

  • Hover效果：鼠标悬停时线条会变粗、变亮，并显示发光效果
  • 流动动画：鼠标悬停时显示流动的光点，表示数据流向
  • 标签缩放：悬停时标签会放大，但不会阻挡线条的hover效果
  • 箭头动画：箭头在悬停时会放大，增强视觉反馈
  • 平滑过渡：所有效果都有平滑的过渡动画，提升用户体验
  • 多种状态：支持普通、选中、高亮、禁用等多种状态
  • 类型样式：不同类型的边有不同的颜色主题
  • 性能优化：使用CSS动画，GPU加速，性能优秀

// 配置边的动画效果
const edge = {
  id: 'edge1',
  source: 'node1',
  target: 'node2',
  label: '审批流程',
  style: {
    type: EdgeType.POLYLINE,
    strokeColor: '#2196f3',
    strokeWidth: 2,
    animated: true,           // 启用动画
    animationType: 'flow',    // 流动动画类型
    animationDuration: 1.5    // 动画持续时间
  }
};

// 添加边到流程图
flowChart.addEdge(edge);

// 边会自动具有以下效果：
// 1. 鼠标悬停时变亮、变粗
// 2. 显示流动的光点动画
// 3. 标签放大但不阻挡hover
// 4. 箭头缩放动画
// 5. 发光效果
